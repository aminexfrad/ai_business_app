# 🧠 AI Business Intelligence — دليل بالعربية

منصة **MLOps** (عمليات تعلم الآلة) بواجهة ويب **Streamlit**، وتتبع التجارب عبر **MLflow** على **DagsHub**، وتحليل بالذكاء الاصطناعي عبر **Google Gemini**.

هذا الملف يساعدك على **فهم التطبيق** و**استخدامه** و**عرضه** أمام اللجنة أو المدرس.

---

## 1. ما هو هذا التطبيق؟

التطبيق يحوّل **بيانات الأعمال** إلى **تنبؤات ذكية** مع تتبّع احترافي لكل تجربة تعلم آلي.

| الوظيفة | الشرح |
|---------|--------|
| **استكشاف البيانات** | إحصائيات ورسوم بيانية وعلاقات بين المتغيرات |
| **تدريب النماذج** | مقارنة عدة نماذج ML على نفس البيانات |
| **MLOps** | حفظ المعاملات والمقاييس والنماذج في MLflow (DagsHub) |
| **التنبؤ** | سطر واحد أو ملف CSV كامل |
| **تحليل بالذكاء الاصطناعي** | أسئلة بالعربية أو الفرنسية لـ Gemini |
| **سجل العمليات** | تاريخ التدريب والتنبؤات وطلبات Gemini |

## 3. مجموعات البيانات الأربع

تختار المجموعة من **الشريط الجانبي الأيسر**.

| المجموعة | النوع | المتغير المستهدف | الاستخدام التجاري |
|----------|-------|------------------|-------------------|
| 🏢 **طاقة المباني** | انحدار (رقم) | `energy_consumption` | توقع استهلاك الطاقة |
| 🍽️ **إيرادات المطعم** | انحدار | `monthly_revenue` | توقع الإيراد الشهري |
| 🔍 **كشف الاحتيال** | تصنيف (0/1) | `fraud` | هل المعاملة احتيالية؟ |
| 📺 **إلغاء الاشتراك** | تصنيف | `subscription_cancelled` | هل سيُلغي المشترك اشتراكه؟ |

## 4. النماذج الثمانية

**للانحدار** (توقع رقم):
- Linear Regression
- Random Forest Regressor
- Gradient Boosting Regressor
- XGBoost Regressor

**للتصنيف** (فئة أو نعم/لا):
- Logistic Regression
- Random Forest Classifier
- Gradient Boosting Classifier
- XGBoost Classifier

بعد التدريب، **نموذج البطل Champion** 🏆 هو الأفضل:
- انحدار → أعلى **R²**
- تصنيف → أعلى **F1-score**

## 5. التبويبات الستة

### 🏠 العرض التقديمي (Présentation)
- صفحة البداية للعرض
- شرح المشكلة والبنية والتقنيات
- ملخص كل مجموعة بيانات

**للعرض:** ابدأ من هنا.

---

### ⚙️ التدريب و MLflow
- معاينة البيانات والإحصائيات
- اختيار النماذج للتدريب
- زر: **« Lancer l'entraînement & Tracker avec MLflow »**

**ما يحدث تلقائياً:**
1. تنظيف البيانات (قيم ناقصة، ترميز الفئات)
2. تقسيم 80٪ تدريب / 20٪ اختبار
3. تدريب كل نموذج وحفظه في MLflow
4. اختيار البطل وحفظه (`champion_model.pkl`)
5. تسجيل البطل في Model Registry

**للعرض:** أهم تبويب — أظهر جدول المقارنة والرسم البياني.

---

### 🎯 تنبؤ لسطر واحد
- إدخال القيم يدوياً
- زر **« Prédire »**
- النتيجة + احتمالات (للتصنيف)
- تفسير اختياري من Gemini

**شرط:** يجب تحميل نموذج البطل أولاً.

---

### 📦 تنبؤ دفعة (Batch CSV)
- رفع ملف CSV
- تحميل قالب (template)
- تنبؤات لكل الصفوف + تصدير النتائج

**للعرض:** سيناريو شركة ترسل ملف عملاء للتقييم.

---

### 🤖 تحليل بالذكاء الاصطناعي
- أسئلة باللغة الطبيعية عن البيانات
- أسئلة جاهزة سريعة
- يستخدم **Gemini** (`gemini-2.0-flash`)

**شرط:** مفتاح `GOOGLE_API_KEY` مع رصيد API.

---

### 📊 لوحة المعلومات (Dashboard)
- مؤشرات KPI
- رسوم: توزيع الهدف، ارتباط، scatter
- مقارنة النماذج (بعد التدريب)
- **سجل العمليات** (Logs)

## 7. التثبيت والتشغيل

### المتطلبات
- Python 3.11+
- حساب DagsHub + token
- مفتاح Google Gemini

### التثبيت

```bash
cd ai_business_app
pip install -r requirements.txt
```

### متغيرات البيئة

أنشئ ملف `.env`:

```env
DAGSHUB_USERNAME=اسم_المستخدم
DAGSHUB_TOKEN=الرمز_السري
DAGSHUB_REPO_NAME=ai_business_app
GOOGLE_API_KEY=مفتاح_gemini
```

على **Streamlit Cloud**: نفس القيم في **Settings → Secrets**.

### تشغيل التطبيق

```bash
streamlit run app.py
```

افتح: **http://localhost:8501**

## 8. سيناريو العرض المقترح (5–10 دقائق)

1. **العرض التقديمي** — المشكلة والبيانات الأربع (30 ثانية)
2. **الشريط الجانبي** — اختر *كشف الاحتيال* أو *طاقة المباني*
3. **التدريب** — اختر 2–4 نماذج → اضغط التدريب (2–3 دقائق)
4. أظهر **البطل** 🏆 والجدول ورابط **DagsHub MLflow**
5. **تنبؤ لسطر واحد** — أدخل قيماً → **Prédire**
6. **لوحة المعلومات** — رسم واحد أو اثنان
7. *(اختياري)* **Gemini** — سؤال تجاري
8. **السجلات (Logs)** — إظهار التتبع MLOps

## 9. مفاهيم MLOps للمناقشة

| المصطلح | الشرح المبسّط |
|---------|----------------|
| **MLflow** | دفتر يسجّل كل تجربة ML |
| **DagsHub** | استضافة بعيدة لـ MLflow + Git |
| **Run** | تشغيل واحد لتدريب نموذج |
| **Experiment** | مجموعة runs (مثلاً: dataset + مهمة) |
| **Champion** | أفضل نموذج للاستخدام في التنبؤ |
| **Model Registry** | كتالوج النماذج بإصدارات |
| **Pipeline** | سلسلة: تنظيف → نموذج → تنبؤ |

## 10. المقاييس المعروضة

**انحدار (Regression):**
- **RMSE** — خطأ أقل = أفضل
- **MAE** — متوسط الخطأ المطلق
- **R²** — جودة النموذج (قريب من 1 = ممتاز) → يختار البطل

**تصنيف (Classification):**
- **Accuracy** — نسبة الصواب
- **F1** — توازن الدقة والاستدعاء → يختار البطل
- **Precision / Recall / AUC**

## 11. حل المشاكل الشائعة

| المشكلة | الحل |
|---------|------|
| « No champion loaded » | درّب في تبويب التدريب، أو اضغط *Load Champion from MLflow* |
| خطأ MLflow | تحقق من `DAGSHUB_USERNAME` و `DAGSHUB_TOKEN` |
| خطأ Gemini 404 | النموذج محدّث إلى `gemini-2.0-flash` |
| خطأ Gemini 429 | نفاد الحصة — انتظر أو فعّل الفوترة في Google AI |
| ملف CSV غير موجود | شغّل التطبيق من مجلد المشروع |
